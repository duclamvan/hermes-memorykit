# Changelog

All notable changes to Hermes MemoryKit use semantic versioning.

## v0.2.0 - 2026-05-19

- Renamed the project to **Hermes MemoryKit** with the memorable repo slug `hermes-memorykit`.
- Added the Hermes-native plugin wrapper:
  - `memory_stack_status`
  - `memory_stack_route`
  - `memory_stack_focus_brief`
  - `memory_stack_regress`
- Added `scripts/install_hermes_plugin.py` for copying the wrapper into a Hermes profile.
- Added `docs/skills-and-memory.md` with the skills-vs-memory promotion policy.
- Added plugin wrapper tests and install verification.

## v0.1.0 - 2026-05-19

- Initial public implementation kit.
- Added architecture docs, config templates, entity graph extraction, hybrid router, focus brief builder, retrieval regression, stack verification, nightly maintenance, and CI.
